# EEP 153 – Project 3: Food Demand and Nutrition in Mali

## 📌 Overview
This project analyzes household food consumption and nutrition in Mali using survey data. The goal is to estimate Engel curves, examine nutritional outcomes, and simulate counterfactual scenarios related to food demand and prices.

---

## 📊 Data
The analysis uses household-level data from Mali, including:

- Food expenditures (2021–2022)
- Household characteristics
- Nutritional content and food prices

All datasets are stored in the `data/` folder.

---

## 📈 Methods
The project includes:

- Estimation of **Engel curves** to understand how food demand changes with income
- Analysis of **nutritional intake**
- Simulation of **counterfactual scenarios** to evaluate changes in consumption

All analysis is conducted in Python using Jupyter Notebook.
